## DESIGN SPECIFICATIONS

### Colors
- **Primary (Orange):** #FF6F00
- **Secondary (Navy):** #0F1A2F
- **Background (White):** #FFFFFF
- **Text (Dark Gray):** #1A1A1A
- **Light Gray (borders/lines):** #E0E0E0

### Typography
- **Headers:** Bold, 18-24pt, Navy #0F1A2F
- **Subheaders:** Semi-bold, 14-16pt, Orange #FF6F00
- **Body Text:** Regular, 11-12pt, Dark Gray #1A1A1A
- **Instructions:** Regular, 10-11pt, Medium Gray #666666

### Layout Guidelines
- **Margins:** 20mm (0.75") on all sides
- **Spacing:** 10-15mm between sections
- **Borders:** 2pt, Light Gray #E0E0E0 or Orange #FF6F00
- **Rounded corners:** 8px on boxes

## EDITABLE vs. PRINTABLE

### Editable PDF (Recommended)
- Users can type directly into form fields
- Create fillable PDF using Adobe Acrobat or online tools
- More user-friendly for digital users

### Printable PDF
- Users print and fill by hand
- Easier to create
- Better for tactile learners

**Suggestion:** Create both versions
