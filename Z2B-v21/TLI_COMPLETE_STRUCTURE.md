## Hybrid Distribution Formula

### How It Works:

1. **Calculate Quarterly Pool**:
   ```
   Quarterly Pool = 10% of Total Member TeamPV (3 months combined)
   ```

2. **Distribute Minimum Guarantees**:
   - Each qualified builder receives their level's minimum guarantee first
   - Example: Level 10 builder gets R2,500,000 minimum

3. **Calculate Remaining Pool**:
   ```
   Remaining Pool = Quarterly Pool - All Minimum Guarantees Paid
   ```

4. **Distribute Pool Bonus by Points**:
   - Each level has a point value (1 to 512)
   - Total all points from all qualified builders
   - Divide remaining pool by total points to get "per point value"
   - Each builder receives: (Their Level Points √ó Per Point Value)

5. **Final Payment**:
   ```
   Builder's Total = Minimum Guarantee + Pool Bonus
   ```
