# Z2B Admin Backend

Z2B Legacy Builders Admin Backend API

## Tech Stack
- Node.js + Express
- MongoDB Atlas
- JWT Authentication
- Yoco Payment Integration

## Environment Variables Required
- MONGODB_URI
- JWT_SECRET
- ADMIN_USERNAME
- ADMIN_PASSWORD
- YOCO_SECRET_KEY

## Default Admin Credentials
- Username: admin
- Password: Admin@Z2B2024!

## Endpoints
- POST /api/auth/login
- POST /api/auth/register
- GET /api/users
- POST /api/tiers/checkout
- And more...
